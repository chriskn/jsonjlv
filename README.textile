h3. Information

JSONJVL is a JSON to Log4j converter for logging evenets which works with the Java logging viewer [JVL](https://github.com/rdiachenko/jlv) . JSONJVL is available as a plugin for Eclipse IDE.


h3. Requirements:
* Eclipse 4.x
* Java 1.7
