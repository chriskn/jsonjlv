h1. JsonJLV

h3. Information

JsonJLV is a JSON to Log4j converter for logging events which works with the Java logging viewer (https://github.com/rdiachenko/jlv) . JsonJLV is available as a plugin for Eclipse IDE.


h3. Requirements:
* Eclipse 4.x
* Java 1.7
